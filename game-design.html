Game Design

Games have been near and dear to my heart for a long time.
First game I've ever played was an emulated "Prince of Persia" on a family computer.
Since then I've played many more games, with my favourite genres being Roguelikes and RPGs.
My favourite games include "Undertale", "Binding of Isaac", "Limbus Company",
"Megaman Battle Network 5" and "Dark Souls".
The first game that made me want to create games was "Spore" with its mission creator.
Since then I've been learning game design and development in multiple languages and engines.
Below you can find some of the games I've worked on, or you can see them all at crystalclod.itch.io/.

Game Design

This Will Be Sports In 1985 (2020)

Game made for the "Mix and Game Jam 2020", and featured by the event organizer in the top games of the jam here.
I've worked on managing this project, as well as programming and dealing with shaders and particle effects.
Game was made in Unity. You can play it here: crystalclod.itch.io/this-will-be-sports-in-1985.

Anticipation (2020)

Game made for the "Stay Safe Jam" that happened during the early stages of the COVID pandemic.
It was the first time I've managed a team for a game, and I've also done all the code for it.
Game was made in Unity. You can play it here: crystalclod.itch.io/anticipation.

Dead Air (2025)

Game made for "PolyJam 2025".
In this jam I was responsible for 3D modeling, shaders, sound effects, particle systems and part of the code.
Game was made in Unity. You can play it here: jasonnumberxiii.itch.io/dead-air.
